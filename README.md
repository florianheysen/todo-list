# La Todo-List

Ajoutez et supprimez vos tâches sur une Todo-list sobre et colorée.

## Utilisation

N'oubliez pas la commande ci-dessous avant de lancer le projet. Elle permet d'installer les modules externes à l'application.

```bash
npm install
```
